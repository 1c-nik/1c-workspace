# Проект: [НАЗВАНИЕ]

## Конфигурация базы

- **Платформа:** 1С 8.3
- **Конфигурация:** [ERP / УТ / БП / УПП / другое]
- **Живая база:** доступна через MCP-сервер `onec-[псевдоним]` (см. `.mcp.json`)

## Структура проекта

```
[псевдоним]\
├── mainConf\                  ← выгрузка типовой конфигурации (read-only)
├── metaData\                  ← индекс базы
│   ├── objects.csv            ← словарь всех объектов
│   └── <Тип>_<Имя>.md        ← структура по каждому объекту
├── extensions\                ← активная разработка расширений (CFE)
├── tasks\
│   ├── active\                ← текущие задачи T-XXXX_название.md
│   └── archive\               ← завершённые задачи
├── _drafts\                   ← черновики, наброски кода
└── .mcp.json                  ← подключение живой базы
```

## Правила проекта

- `mainConf\` — только чтение, не модифицировать
- `metaData\` — читать перед работой с любым объектом; не парсить XML из `mainConf\`
- Все изменения — только через расширения (CFE) в `extensions\` или внешние EPF/ERF
- Задачи в формате `tasks\active\T-XXXX_название.md`
